2. Type in address bar: about:debugging#/runtime/this-firefox

4. Navigate to: e:\Projects\scripts\reddit-to-x-extension

### Chrome/Edge (if needed):


- Open Edge browser with Selenium
- Wait for Reddit URLs from the extension
